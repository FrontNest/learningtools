This subpage is for opening a .csv file from your PC and searching the data.
It loads the first 2000 rows, displays columns B:G,
and allows you to search based on keywords in column A.
The operation is performed by JavaScript.
This small tool can be useful when you need to search through a lot of small information.
For example, for customer service telephone dispatchers,
this format of the knowledge base can be useful during calls.
